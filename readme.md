# How to install
* Install `Visual Studio 2017 Community edition`
* Install `exuberant ctags`
* Install `python3`
* Install vim
* Install vim-plug from `https://github.com/junegunn/vim-plug`
* Copy vimfiles to ~/
* Run gVim
* `:PlugInstall`
* In vimfiles/plugged/YouCompleteMe:
`python3 install.py --clangd-completer --msvc=15`
